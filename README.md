# Gesture-Translating-System-Using-STM32

  This program is for the gesture translating gloves. The system take input finger movement through flex sensors, hand movement and position via MPU to indicate the hand gesture. Then, the input will be gather as a chain of gesture and the algorithm will compare that action with the action library to indicate what is the gesture the user doing.

  The product is aimed for the blind and deaf people who has the difficult in communicating with the normal people.
